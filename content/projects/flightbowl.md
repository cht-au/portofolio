FlightBowl was built as a group project for our **Database Concepts** course. 

## 📌 Project Goal 
Our goal was to design an app that simulates a flight booking and management system to coordinate flights for Notre Dame marching band members across different airports for bowl games. Managing group logistics on this scale requires robust data handling to ensure everyone gets to the game on time.

---

## 🛠️ System Architecture & Tech Stack
We built a full-stack application leveraging the following technologies:
* **Frontend:** React
* **Backend:** Python / Flask / SQLAlchemy (ORM)
* **Database:** MySQL

---

## 🚀 Results
We successfully implemented a fully functional web app that achieves our core tracking requirements:

* **Asynchronous Flight Scraping:** Integrated asynchronous scraping to dynamically find and pull available flights into the system.
* **Roster & Deviation Tracking:** The app reliably keeps track of different band members and handles any specific flight deviations each individual might have, ensuring flexible logistics management.
